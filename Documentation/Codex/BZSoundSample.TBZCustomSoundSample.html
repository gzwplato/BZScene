BZSoundSample

Déclaration

type TBZCustomSoundSample = class(TBZCustomDataFile)

Description

Classe de base représentant un échantillon sonore et les informations relative à sa qualité.

Cette classe disposes de quelques fonctions utiles pour la convertion des ondes sonores et pour récupérer les "peaks" à un temps donnés.

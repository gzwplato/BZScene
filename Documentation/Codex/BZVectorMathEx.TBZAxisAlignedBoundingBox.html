Description

Décrisption d'une boite englobante alignée en 3D (AABB)

Note :
'Le calcul de base sous-jacent du lancer de rayons est celui de l'intersection d'une ligne avec les surfaces d'un objet.
Est une méthode pour effectuer ce calcul, pour une nouvelle et puissante classe d'objets, ceux définis en balayant une sphère de rayon variable le long d'une trajectoire 3D.
  Lorsque des polynômes sont utilisés pour le paramétrage du centre et du rayon de la sphère,
   Le problème d'intersection se réduit à l'emplacement des racines d'un polynôme. '
Source : https://www.sciencedirect.com/science/article/pii/009784938590055X

Description

Contient un objet TBZParallelThread qui permet d'exécuter des opérations en parallele facilement grâce au multithreading.

Historique :

  • Creation : 24/06/2019

  • Mise à jour :

Notes :
Lazarus dispose de cette possibilité nativement, mais est un peu plus compliqué à mettre en place. cf : https://wiki.freepascal.org/Parallel_procedures/fr

Inspirer du code source de Paul Toth pour Delphi http://lookinside.free.fr/delphi.php?Multithreading)

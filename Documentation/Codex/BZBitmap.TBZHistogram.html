Déclaration

type TBZHistogram = class(TBZCustomOwnerBitmap)

Description

Objet permettant le calcul et la manipulation de l'histogramme d'une image.
D'après les codes source de Earl F. Glynn entre autres.

Plus d'informations :

  • http://www.tsi.enst.fr/pages/enseignement/ressources/mti/egal-histo/rapport.htm

  • https://en.wikipedia.org/wiki/Adaptive_histogram_equalization

  • https://en.wikipedia.org/wiki/Normalization_(image_processing)

  • https://stackoverflow.com/questions/56905592/automatic-contrast-and-brightness-adjustment-of-a-color-photo-of-a-sheet-of-pape

  • http://www.eyemaginary.com/Rendering/AutomaticHistogramWarping.pdf

  • https://openclassrooms.com/fr/courses/1490316-introduction-a-la-vision-par-ordinateur/1491061-etirement-et-egalisation-dhistogramme

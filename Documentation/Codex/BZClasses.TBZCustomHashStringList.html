Déclaration

type TBZCustomHashStringList = class(TBZCustomDataFile)

Description

Classe optimisée pour utiliser une liste de chaine de caractères avec la possibiliter d'associer un nom, un d'objets et ou un Tag.
Elle intègre une recherche basé sur le "Haschage".
Ce qui fait que la recherche est un simple test de correspondance.
La liste n'a donc pas besoin d'être triée pou effectuer un recherche rapide..

Procedure DirectSort;

La procedure de trie ici n'utilise pas la fonction de base par défaut qui utilise l'algorithme QuickSort. A la place le trie est effectué par l'algorithme "A-Sort" qui est 1000x plus rapide. Cet algorithme est basé sur un article de Lionel Defosse parue dans le magazine pascalissisme n°59. ( https://ldelafosse.pagesperso-orange.fr/Sort/Papers/Asort.htm )
Le trie est effectué uniquement dans DirectList dans un soucis de performance.
La Liste de l'objet n'est pas impacté.

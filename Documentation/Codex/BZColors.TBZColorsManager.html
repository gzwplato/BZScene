Unité

BZColors

Déclaration

type TBZColorsManager = class(TBZCadenceAbleComponent)

Description

Objet servant à manipuler une palette de couleur en "temp réel".

  • Avancer/reculer, boucler

  • Chargement de palette spéciales prédéfinies

  • Creation de couleurs dans la palette

  • Creation de degradés depuis une liste de couleurs

  • Lecture, écriture da la liste de couleurs depuis, vers un fichier

  • TBZColorsManager

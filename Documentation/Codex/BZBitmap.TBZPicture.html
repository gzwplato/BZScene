BZBitmap

Déclaration

type TBZPicture = class(TBZPersistentObject)

Description

Classe d'aide surtout utile pour l'integration d'un bitmap dans un composant (non) visuel.
Elle permet également de définir le format les options adequates pour la sauvegarde d'une image.

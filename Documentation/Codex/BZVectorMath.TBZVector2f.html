function ManhattanDistance(constref A : TBZVector2f) : Single;

Retourne la distance de Manhattan.
Note : La distance de Manahattan est la distance de "marche" entre deux points.
Aucun déplacement angulaire n'est autorisé. (applicable uniquement pour des mouvements horizontaux et verticaux)

function ChebyshevDistance(constref A : TBZVector2f) : Single;

Retourne la distance de Chebyshev. Similaire à ManhattanDistance.
Sauf que le mouvement diagonal est autorisé.
Note : Cette fonction est aussi appelée "distance de l'échiquier" car elle correspond au nombre de mouvements qu'une pièce du roi doit effectuer pour se déplacer entre deux points.

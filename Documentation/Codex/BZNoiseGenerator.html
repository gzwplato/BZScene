Description

Contient des classes d'aide à la génération de bruit.

tel que bruit blanc, bruit blanc additif gaussien, bruit de perlin, bruit fractal, bruit cellulaire etc...
Ces classe permettent la creation de divers signaux ainsi que la génération de texture.

TBZCoordinatesStyle = (...);

TBZCoordinatesStyle : Identifie le type de données stockées au sein d'un TBZCustomCoordinates. csPoint2D : a simple 2D point (Z=0, W=0) csPoint : un point (W=1) csVector : un vecteur (W=0) csUnknown : aucune contrainte

  • CsVector

Déclaration

type TBZRawImageData = class(TBZUpdateAbleObject)

Description

Classe "semi-abstraite" décrivant les informations de données brute d'un bitmap (image).
On stocke ici sa description, ainsi que la palette de couleur "brute" si elle existe.
Les informations sont à mettre à jour manuellement. Lors du chargement d'un fichier par exemple ou lors de la création d'une nouvelle image.

Les données des brutes, ne sont pas stockées. Ici, car dans BZScene, lors du chargement d'un bitmap celui-ci est automatique convertit vers le format RGBA ou BGRA suivant l'OS. (cf : TBZBitmap ).
Une fois l'image chargée et la description remplie. On peux modifier, certain paramètres comme les propriétés PixelFormat, ou ColorFormat pour l'export des données. Celles-ci seront alors convertient depuis les données contenue dans TBZBitmap.

Unité

BZBitmap

Déclaration

type TBZBitmapMask = class(TBZBitmap)

Description

Classe pour la gestion d'un Masque.
Couleur blanche (clrWhite) = Pixel visible
Couleur noir ou transparente (clrBlack, clrTransparent) = Pixel Invisible
Cette classe est dérivée de TBZBitmap afin d'avoir accès aux différents filtres et procedures de transformations.

Unité

BZBitmap

Déclaration

type TBZBitmapCanvas = class(TBZCustomCanvas)

Description

Regroupe les fonctions pour dessiner dans un TBZBitmap. - point, ligne, cercke, éllipse, rectangle, triangle, polygone, arc, chord, bezier, spline, remplissage, texte....
Avec ou sans adoucissement, remplissage uniforme, patterne, degradé, texture, options pour l'aspect des contours...

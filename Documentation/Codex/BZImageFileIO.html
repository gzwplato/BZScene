Description

La classe décrite ici, "TBZCustomImageFileIO" est l'ancêtre des classes servant à lire ou écrire les différents formats d'image supportés (BMP, TGA, PCX,...).
Elle est un pont direct avec notre gestionnaire de bitmap "TBZCustomBitmap".

Cette classe est avant une classe d'aide à la lecture et l'écriture des données.
Elle inclue des fonctions utiles pour la lecture des pixels dans un format donné RGB 16bits, RGB 24bits, BGRA,... ; pour créer la palette de couleur issue du fichier.
Ou bien encore pour décompresser, ou décoder les données suivant un schéma fournit.

Elle inclut également le support d'informations complémentaires lors de la lecture d'un fichier. Par exemple les données EXIF des fichiers JPEG.

Description

Descrition d'angles d'Euler

Les angles d'Euler sont trois angles pour décrire l'orientation d'un corps rigide par rapport à un repère fixe.
    Ils peuvent également représenter l'orientation d'un référentiel mobile en physique. Ou l'orientation d'une base générale en algèbre linéaire tridimensionnelle. Est utilisé par le type TBZQuaternion.

A RETENIR :

  • X = Roll = Bank = Tilt

  • Y = Yaw = Heading = Azimuth

  • Z = Pitch = Attitude = Elevation

Source : https://en.wikipedia.org/wiki/Euler_angles

Description

Prise en charge des Images au format "Portable pixmap"

Spécifications :

  • Méthode de compression : Aucune

  • Nombre de couleurs : 1, 8 bits format RGB, 24 bits, RGBA 32 bits

  • Supporte plusieurs images : Non, une seule image dans un même fichier

  • Format des nombres : texte et binaire

  • Auteur : ???

  • Extensions : *.pbm, *.pgm, *.pnm, *.ppm, *.pam, *.pfm, *.rpbm, *.rpgm, *.rppm, *.ppma

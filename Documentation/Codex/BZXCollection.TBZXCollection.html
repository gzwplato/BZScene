Unité

BZXCollection

Déclaration

type TBZXCollection = class(TPersistent)

Description

Contient une liste d'objets TBZXCollectionItem.

    Cette classe ressemble beaucoup à un TCollection, mais est polymorphe.
Il y existe une version très dépouillée d'un TObjectList propriétaire et de la persistance     classes (XClasses & XLists), si les droits d'auteur sont partiellement levés     sur les originaux, je vais baser ce code sur eux car ils sont beaucoup plus rapides     que les listes et les mécanismes de persistance standard.
    TBZXCollection supporte le polymorphisme et une compatibilité descendante complète.

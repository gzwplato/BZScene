Description

Unité regroupant plusieurs classes de gestion d'algorithmes de compressions.

Les algorithmes supportés ici s'utilisent principalement avec des données provenent d'images (BMP, TGA, PCX etc...)

Les méthodes supportées :

  • RLE_BMP, RLE_TGA, RLA_PCX, RLE_PackBits

  • LZW, Huffman, thunder

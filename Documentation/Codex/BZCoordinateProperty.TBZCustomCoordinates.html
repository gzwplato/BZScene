Déclaration

type TBZCustomCoordinates = class(TBZUpdateAbleObject)

Description

TBZCustomCoordinates :  Cette classe est fondamentalement un conteneur pour un TBZVector, permettant une utilisation correcte de l'éditeurs de propriétés et l'édition dans l'EDI.     les méthodes de manipulation sont seulement minimes.     Gère les valeurs par défaut dynamiquememnt pour l'enregistrement dans le fichier de ressources.

Propriétés

property Style: TBZCoordinatesStyle read FStyle write FStyle;

Style : Identifie les styles de coordonnées.       La propriété n'est pas persistante. csUnknown par défaut. Il devrait être géré par l'objet propriétaire uniquement (en interne).       Il est utilisé par TBZCustomCoordinates pour les vérifications internes d ''assertion'       détecter les «abus» ou les «malentendus» que Le système de coordonnées homogène implique.

property AsVector: TBZVector read FCoords write SetAsVector;

AsVector : Les coordonnées sont vues comme un vecteur.       L'attribution d'une valeur à cette propriété déclenchera des événements de notification,       Si vous ne le souhaitez pas, utilisez plutôt DirecVector.

property AsAffineVector: TBZAffineVector read GetAsAffineVector write SetAsAffineVector;

AsAffineVector : Les coordonnées sont vues comme un vecteur affine.       L'attribution d'une valeur à cette propriété déclenchera des événements de notification,       Si vous ne le souhaitez pas, utilisez plutôt DirecVector.       Le composant W est automatiquement ajusté en fonction du style.

property AsPoint2D: TBZVector2f read GetAsPoint2D write SetAsPoint2D;

AsPoint2D : Les coordonnées considérées comme un point 2D.       L'attribution d'une valeur à cette propriété déclenchera des événements de notification,       Si vous ne le souhaitez pas, utilisez DirecVector à la place

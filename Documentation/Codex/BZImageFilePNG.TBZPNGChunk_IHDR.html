IHDR chunk format - En-tête PNG.

Aperçu

Champs

Width: LongWord;
Height: LongWord;
BitDepth: Byte;
ColorType: Byte;

Description

Champs

Width: LongWord;

Image width

Height: LongWord;

Image height

BitDepth: Byte;

Bits par pixel ou bits par canal (pour "TrueColor")

ColorType: Byte;

0 = grayscale, 2 = truecolor, 3 = palette,

Compression: Byte;

Compression type: 0 = ZLib

Filter: Byte;

Filtre de prediction utilisé avant la compression

Interlacing: Byte;

Entrelacé: 0 = non int, 1 = Adam7

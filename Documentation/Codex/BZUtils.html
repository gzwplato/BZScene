Description

L'unité BZUtils contiens des routines utiles dans différents domaines tel que les nombres, les chaine de caractères, la manipulation de bits, le calcul de CRC32 etc...
Certaines routines ont étés optimisées gràce à l'utilisation de l'assembleur.

————————————————————————————————————-

Historique :

  • Creation : 24/02/2019

————————————————————————————————————-

Notes :
Par défaut, certaines routines utilisent l'assembleur.
  Pour désactiver l'utilisation de l'assembleur, commentez 'USE_ASM_OPTIMIZATIONS' dans bzscene_options.inc

Dépendances : BZSceneStrConsts, BZSystem

Déclaration

type TBZBitmapLayerItem = class(TBZUpdateAbleObject)

Description

Objet pour stocker des images supplémentaire dans un TBZCustomBitmap.
Utile dans le cas d'images animées (gif, apng) ou de fichier multi-images (tiff, mng, psd etc...) Peux également être utilisé dans la gestion de calque comme photoshop ou gimp.

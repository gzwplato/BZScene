function Rotate(constref axis : TBZVector; angle : Single):TBZVector;

Effectue une rotation du point selon un axe arbitraire et d'un angle (en radians)

Le système de coordonnées utilisé ets "Main droite".
Les rotations positives sont dans le sens inverse des aiguilles d'une montre avec l'axe positif vers vous.
Les rotations positives sont vues dans le sens horaire depuis l'origine le long de l'axe positif :

  • Orientation de l'axe pour visualiser les positifs dans le quadrant supérieur droit [sous forme d'axes de graphique]

  • Avec + Z pointé vers vous (vers l'écran), +X est à droite, +Y est vers le haut

  • Avec + Y pointant vers vous, +Z est à droite, +X est vers le haut

  • Avec + X pointant vers vous +Y est à gauche +Z est vers le haut

function PointProject(constref origin, direction : TBZVector4f) : Single;

Projection du point sur la ligne définie par "Origin" et "Direction".

Effectue un produit scalaire : DotProduct((p - origine), direction) qui, si la direction est normalisée, calcule la distance entre l'origine et la projection deu point sur la ligne (origine, direction).

function MoveAround(constref AMovingObjectUp, ATargetPosition: TBZVector; pitchDelta, turnDelta: Single): TBZVector;

Modifie la position du vecteur (caméra) pour le déplacer autour de sa cible.
      Cette méthode permet de mettre en œuvre rapidement les commandes d'une caméra.
Note : Angle deltas est en degré.

Astuce :
Les coordonnées du parent vecteur (caméra) doivent être une "identité".
Faire du vecteur de la camera un enfant d'un objet, et en déplacant simplement la cible, permet de changer l'angle de vue. (style FPS)

function FaceForward(constref A, B: TBZVector4f): TBZVector4f;

Retourne une normale telle quelle si le vecteur de position dans l'espace occulaire d'un sommet pointe dans la direction opposée d'une normale géométrique, sinon renvoie la version inversée de la normale.

Le vecteur courrant représente le vecteur de la normale pertubée.
Le vecteur A représente un vecteur d'incidence (généralement un vecteur de direction de l'œil vers un sommet)
Le vecteur B représente le vecteur normal géométrique (pour certaines facettes, la normale péturbée est identique).

Plus d'informations :

  • http://developer.download.nvidia.com/cg/faceforward.html

  • https://www.khronos.org/registry/OpenGL-Refpages/gl4/html/faceforward.xhtml

function Saturate: TBZVector4f;

Renvoie le plus petit entier non inférieur à un scalaire ou à chaque composante vectorielle.

     Retourne la valeur de la saturation dans la plage [0,1] comme suit:

  1. Retourne 0 si le vecteur courrant est plus petit que 0

  2. Retourne 1 si le vecteur courrant est plus grand que 1

  3. Dans les autres cas retourne les valeurs courrantes.

Plus d'informations : http://developer.download.nvidia.com/cg/saturate.html )

function SmoothStep(ConstRef A,B : TBZVector4f): TBZVector4f;

Interpolation douce entre deux valeurs basé sur le troisième paramètre.

  • Retourne 0 si Self < A < B ou que Self > A > A

  • Retourne 1 si Self < B < A ou que Self > B > A

  • La valeur interpolée retournée est comprise dans l'intervalle 0.0 et 1.0 pour le domaina [A,B].

Si A = Self alors, la pente de Self.smoothstep(a, b) et b.smoothstep(a, b) est nulle.

Plus d'information :

  • http://developer.download.nvidia.com/cg/smoothstep.html

  • https://www.khronos.org/registry/OpenGL-Refpages/gl4/html/smoothstep.xhtml

Description

Prise en charge des Images au format X PixMap.

Spécifications :

  • Méthode de compression : Aucune

  • Nombre de couleurs : Format RGB, 24 bits ou 32 bits RGB(A)

  • Supporte plusieurs images : Non, une seule image dans un même fichier

  • Format des nombres : Texte

  • Auteur : Colas Nahaboo et Daniel Dardailler.

  • Extensions : *.xpm ( *.xbm)

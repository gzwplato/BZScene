Description

Prise en charge des Images au format WEBP proposé par Google.

Spécifications :

  • Version librairie : 1.0.3

  • Méthode de compression : VP8 or VP8L

  • Nombre de couleurs : 24 / 32 bits : (RGB/A, BGR/A, YUV)

  • Supporte plusieurs images : Non

  • Format des nombres : -

  • Auteur : Google

  • Extensons : *.webp; ( *.webm = video)

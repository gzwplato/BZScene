Unité

BZControlClasses

Déclaration

type TBZCustomControl = class(TCustomControl)

Description

Classe abstraite de base pour les composants visuel.
Active le DoubleBuffer sous Windows et désactive l'effacement du fond du "Canvas".
Offre la gestion de l'affichage avec prise en charge de la transparence.

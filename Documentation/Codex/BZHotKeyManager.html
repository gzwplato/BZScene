Description

Contient un composant pour gérer et interagir avec les raccourcis clavier.
TBZHotKeyManager permet d'intercepter les raccourcis clavier depuis le système (Application minimisée) et dans la fenêtre active en cours.

Notes : - Compatible Windows et Linux (Gnome, KDE, XFCE...) - Sous MacOS fonctionne si vous utiliser GTK et X11

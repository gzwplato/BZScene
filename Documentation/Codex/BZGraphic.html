Description

Définit et regroupe les systèmes de couleurs les types de filtres et autres données utiles à la gestion du graphisme.
Contient les classes de base à la gestion des informations sur les formats d'image

  • asmOpaque : Le canal alpha est uniformément réglé sur 255

Description

Contient une bibliothèque de classes mathématique optimisée pour les vecteurs utilisant l'accélération SIMD (SSE, SSE3, SS4, AVX, AVX2)
Elles peuvent être utilisées dans les graphiques 2D / 3D et tout autre application nécessitant des calculs mathématique avec des vecteurs.

Types de vecteurs supportés :

  • Vecteur 2D Integer, Single et Double

  • Vecteur 3D Byte, Integer, and Single

  • Vecteur 4D Byte, Integer, Single

  • Matrice 2D Single

  • Matrice 4D Single Matrix (@TODO Integer ????)

  • Quaternion

  • Plan Homogène

Contient également des fonctions identique au script GLSL / HLSL

Notes :
Remarquez, ici, nous devons redéfinir le mode d'arrondi FPC comme identique à notre code SSE.
  Dans FPC, la 'fonction Round' utilise l'algorithme 'd'arrondissement bancaire'.
  Il ne fait pas un "RoundUp" ou "RoundDown" si le paramètre d'entré est exactement x.50.

Exemples :
Round(2.5) Resultat = 2
Round(3.5) Resultat = 4

Pour plus d'informations voir : https://www.freepascal.org/docs-html/rtl/system/round.html

Alignement des données sur la limite de 16 bits :
Sous x64, pour utiliser les fonctions vectorielle, vous aurez besoin d'ajouter des commandes de préprocesseur pour obtenir des données alignées appropriées. Normalement, ce n'est pas nécessaire sous x86. Mais c'est recommandé pour plus de sécurité.

Dans l'en-tête des unité ajoutez : 

Unit MyUnit;
{$mode objfpc}{$H+}
{$ALIGN 16}
{$CODEALIGN CONSTMIN=16}
{$CODEALIGN LOCALMIN=16}
{$CODEALIGN VARMIN=16}

  1. Pour les constantes :
    Vous devez d'entourer votre variable comme ceci : 

    Const
  {$CODEALIGN CONSTMIN=16}
  cMyVector : TBZVector = (X:0.5;Y:0.5;Z:0.5;W:1.0);
  {$CODEALIGN CONSTMIN=4}

  2. Pour des Variables ou champs à l'intérieur d'une classe :
    Vous devez d'entourer votre variable comme ceci : 

    Type
  TMyClass = Class
  private
    {$CODEALIGN RECORMIN=16}
    FVec : TBZVector;
    FMatrix : TBZMatrix;
    {$CODEALIGN RECORMIN=4}
    FFactor : Single;
  public
    property Vec : TBZVector read FVec write FVec
  end;

  3. Pour des variables dans des fonctions ou procédures :
    Vous devez d'entourer votre variable comme ceci : 

    function MyFunc(A,B,C:Single):Single;
var
  {$CODEALIGN VARMIN=16}
  MyTempVec : TBZVector
  {$CODEALIGN VARMIN=4}
begin
  //...
end;

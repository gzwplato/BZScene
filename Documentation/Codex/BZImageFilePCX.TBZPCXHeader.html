Packed Enregistrement TBZPCXHeader

Description
Hiérarchie
Champs
Méthodes
Propriétés

Unité

BZImageFilePCX

Déclaration

type TBZPCXHeader = packed record

Description

PBZPCXHeader : En-tête fichier PCX - FileID: byte; signature $0A fichiers PCX, $CD fichiers SCR

Aperçu

Champs

Version: byte;
Encoding: byte;
BitsPerPixel: byte;
XMin: word;
YMin: word;
XMax: word;
YMax: word;
HRes: word;
VRes: word;
ColorMap: array[0..15] of TBZColor24;
Reserved: Byte;
ColorPlanes: byte;
BytesPerLine: Word;
PaletteType: word;
Fill: array[0..57] of byte;

Description

Champs

Version: byte;

0: version 2.5, 2: 2.8 avec palette, 3: 2.8 sans palette, 5: version 3

Encoding: byte;

Compression

BitsPerPixel: byte;

nombre de bits par pixel de l'image: 1, 4, 8, 24

XMin: word;

abscisse de l'angle supérieur gauche

YMin: word;

ordonnée de l'angle supérieur gauche

XMax: word;

abscisse de l'angle inférieur droit

YMax: word;

ordonnée de l'angle inférieur droit

HRes: word;

résolution horizontale en dpi

VRes: word;

résolution verticale en dpi

ColorMap: array[0..15] of TBZColor24;

Palette 16 couleurs

Reserved: Byte;

Réservé NOTE : Ici c'est bizarre car normalement reserved est avant colorplane. Il y a inversion à la lecture, mais je ne sais pas pourquoi

ColorPlanes: byte;

Nombre de plans de couleur

BytesPerLine: Word;

Nombre de bits par ligne

PaletteType: word;

Type de palette : 1 = couleur ou N/B, 2 = dégradé de gris, 0 = peux-etre utilisé dans le cas de fichier dont l'en-tête est incorrect

Fill: array[0..57] of byte;

Réservé

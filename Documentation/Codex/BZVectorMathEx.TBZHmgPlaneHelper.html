Méthodes

function Contains(const TestBSphere: TBZBoundingSphere): TBZSpaceContains;

Détermine dans quel espace (TBZSpaceContains) du plan un sphere englobante est contenue

function EvaluatePoint(constref Point : TBZVector) : Single;

Calcule le produit croisé (Crossproduct) entre la normal du plan et le plan à un point donné.
      Cette fonction donne une indication sur la position du point, si le point est dans le demi-espace pointé par le vecteur, le résultat est positif.
      Note : Cette fonction effectue un produit scalaire spatial homogène.
